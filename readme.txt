== Description ==

The Payment Gateway GMOPG for WooCommerce plugin adds the functionality to take GMO Payment Gateway payments on your store of WooCommerce.

= About GMO Payment Gateway =

GMO Payment Gateway is a comprehensive payment platform that offers various online payment methods, such as credit card payment and convenience store payment in Japan.

[GMO Payment Gateway](https://www.gmo-pg.com/)

In order to use GMOPG, you have to purchase the authentication key at the following site.

[WordPress Market](https://www.wpmarket.jp/)

== Installation ==

1. Upload the plugin file in the plugin install page or copy the `wc-gmopg-gateway` directory into your `wp-content/plugins` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it! :)
